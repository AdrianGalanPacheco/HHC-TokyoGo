Enlaces

En esta sección se documentan todas las fuentes de contenido, imágenes, herramientas y librerías utilizadas para la creación de este sitio web.

Fuentes de contenido

El contenido textual de las páginas de detalle se ha elaborado a partir de la información general disponible en las siguientes fuentes de referencia:

  • GetYourGuide - Food in Tokyo
  • Japonpedia - Comer en Tokio
  • Japonpedia - Templos y Santuarios de Tokio

Atribuciones de imágenes y multimedia

Los recursos visuales utilizados en este proyecto provienen de las siguientes plataformas:

  • Imágenes libres de derechos obtenidas de Unsplash.
  • Favicon descargado desde Flaticon.
  • Vídeos incrustados obtenidos de YouTube. Los derechos pertenecen a sus respectivos creadores.

Tecnologías y librerías

Este sitio web ha sido desarrollado utilizando las siguientes herramientas de código abierto:

  • basicLightbox - Librería para galerías modales.
  • FontAwesome - Librería de iconos.
